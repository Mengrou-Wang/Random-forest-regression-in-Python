# Random-forest-regression-in-Python

The purpose of this project is to identify second-hand Ford vehicles and their characteristics or specifications that appeal to consumers over the last few years. Using the data from Kaggle, key recommendations for dealers to maximize profit are:
- To get a good price for a Ford used car, it is important to focus on the cars that are newly produced, with a higher engine size.
- Try to avoid traditional cars such as manual-petrol cars, especially shift from Fiesta to Kuga and Focus and try to find more hybrid, electric cars.
